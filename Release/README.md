## 🇵🇱 Po polsku

### Co robi ten mod?

**Doradca Wieży** to pomocnik wyświetlający podpowiedzi bezpośrednio na ekranie gry. Nie ingeruje w mechanikę – tylko doradza.

#### ⚔️ HUD Walki (F1 – włącz/wyłącz)
- HP i blok Twój oraz wrogów
- Intencje wrogów z wartościami obrażeń
- Ostrzeżenia: **ŚLISKOŚĆ**, **NIEMATERIALNY**, **PANCERZ** (limit dmg)
- Twoje aktywne debuff: Słaby, Podatny, Wątły, Trucizna
- Aktywny **CORRUPTION** – przypomnienie że Skille kosztują 0
- Sugestia najlepszej karty do zagrania z powodem
- Ostrzeżenie **MOŻESZ ZABIĆ TEJ TURY** / **ZGINIE BEZ BLOKU**
- Kolejność zagrywania kart (synergie: Zręczność przed blokiem, Siła przed atakami)

#### 🃏 Doradca Decku (przy wyborze kart po walce)
- Ocena każdej oferowanej karty (★★★ / ★★ / ★ / ✗)
- Powód oceny: synergie, braki w decku, exhaust engine itd.
- Aktualny archetyp decku (Combo, Siłowy, Exhaust Engine, Agresywny...)
- Sugestia karty do ulepszenia

#### 🔧 Doradca Ulepszeń (przy kowal/ognisko)
- Lista kart do ulepszenia z priorytetem i powodem
- Sugestia kart do usunięcia

#### 🗺️ Doradca Ścieżki (na mapie)
- Analiza dostępnych ścieżek (walki, elity, sklepy, odpoczynki)
- Rekomendacja trasy zależnie od HP i stanu decku

#### 📜 Doradca Wydarzeń
- Ocena opcji w wydarzeniach (leczenie, relikwia, ulepszenie, przekleństwo...)
- Obsługuje opisy po angielsku i polsku

### Instalacja
1. Zamknij grę
2. Uruchom `INSTALL.bat` jako administrator (lub zwykły użytkownik)
3. Uruchom grę → Settings → Mods → włącz DoradcaWiezy

### Skróty klawiszowe
| Klawisz | Akcja |
|---------|-------|
| F1 | Włącz/Wyłącz HUD walki |
| F2 | Ustawienia moda (język PL/EN, włącz/wyłącz moduły) |

## 🇬🇧 English

### What does this mod do?

**Tower Advisor** (Doradca Wieży) is an advisor overlay that shows hints directly on screen. It does not change game mechanics – it only gives advice.

#### ⚔️ Combat HUD (F1 – toggle)
- Your HP/Block and all enemies' HP/Block/intent
- Enemy intent with exact damage values
- Warnings: **SLIPPERY**, **INTANGIBLE**, **ARMOR** (damage cap)
- Your active debuffs: Weak, Vulnerable, Frail, Poison
- Active **CORRUPTION** reminder – Skills cost 0 this turn
- Best card suggestion with reason
- **YOU CAN KILL THIS TURN** / **DIE WITHOUT BLOCK** alerts
- Optimal play order (Dexterity before blocks, Strength before attacks)

#### 🃏 Deck Advisor (at card reward screen)
- Rating for each offered card (★★★ / ★★ / ★ / ✗)
- Reason: synergies, deck gaps, exhaust engine detection, etc.
- Current deck archetype (Combo, Strength, Exhaust Engine, Aggro...)
- Upgrade suggestion

#### 🔧 Upgrade Advisor (at smiths/campfires)
- Prioritized upgrade list with reasons
- Cards to consider removing

#### 🗺️ Path Advisor (on map)
- Path analysis (fights, elites, shops, rests, events)
- Route recommendation based on HP and deck state

#### 📜 Event Advisor
- Evaluates event options (healing, relic, upgrade, curse...)
- Handles both English and Polish descriptions

### Installation
1. Close the game
2. Run `INSTALL.bat` (as admin if needed)
3. Launch game → Settings → Mods → enable DoradcaWiezy

### Hotkeys
| Key | Action |
|-----|--------|
| F1 | Toggle combat HUD |
| F2 | Mod settings (PL/EN language, enable/disable modules) |
